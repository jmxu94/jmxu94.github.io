CoffeeBuzz & Yelp API

Bzzz! I'm currently working on an app that utilizes the Yelp API and randomly selects a coffeeshop within a user's location given a set of reasonable specifications. Every time the user hits the "buzz" button, they'll get a randomly generated cafe near their location. This coffeshop is saved in their data so the user won't ever go to the same coffeeshop twice! The goal of this app is to encourage users to explore their surroundings, go outside, and try new places!
